How Follow-up Status Works

Status is calculated automatically from the follow-up date set when the visit was logged.

Done
When: Employee opened ✏️ Edit and set Reminder Status to Done or Sent. Follow-up confirmed complete.
Upcoming
When: Follow-up date is in the future, not marked Done. Scheduled but not yet due.
Due today
When: Follow-up date is today and not marked Done. Action needed today.
Overdue
When: Follow-up date has passed and never marked Done. Number in brackets = days overdue.
No follow-up
When: No follow-up date was set. Visit was informational or no action needed.
To mark Done: Go to All Visits → find the entry → click ✏️ Edit → update Reminder Status to Done in the Google Form.
